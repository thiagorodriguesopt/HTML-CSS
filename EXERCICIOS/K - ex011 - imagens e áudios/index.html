Colocando audio

Para colocar audio usamos a TAG <audio> e depois colocamos algumas atribuições

<audio> src="musica /gta- musica.mp3" controls></audio>

Ou podemos fazer assim caso queremos colocar outro tipos de audio, que são MP3 (mais usado), WAV, OGG:


        <audio controls preload="metadata">
            <source src="musica/gta- musica.mp3" type="audio/mp3">
            <source src="musica/gta- musica.wav" type="audio/wav">
            <source src="musica/gta- musica.ogg" type="audio/ogg">
        </audio >

Os atributos colocados no audio a cima são CONTROLS (para aparecer o controles de auto) e PRELOAD (que serve para informa se a plataforma carrega os dados completo {auto} se carrega apenas as informações basicas {metadata} ou se não é para carregar nada {none})
