Existe várias ref (referencias) para indicar se aquele link é seu ou de outra pessoa, assim nos utilizamos o cod ref, que serve para referenciar se aquele link é seu, e utilizamos os seguintes atributos a ele

  • NEXT: indica que é um link para proxima pag. do proprio documento.
  • PREV: Indica que é uma pag. anterior do documento.
  • AUTHOR: Indica que é um link para o site do autor do artigo.
  • EXTERNAL: Indica que é um link para outro site que não faz parte do site.
  • NOFOLLOW: Indica que é um link para um site não endossado, como um link pago.
